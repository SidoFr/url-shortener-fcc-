FreeCodeCamp URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

try me:

add /short/the url you wish
to this page's url.
don't forget http(s)://!

examples:
http://localhost:7777/short/https://www.google.com
http://localhost:7777/short/https://www.instagram.com
http://localhost:7777/short/https://www.qwant.com
http://localhost:7777/short/www.twitter.com(wrong!!)
